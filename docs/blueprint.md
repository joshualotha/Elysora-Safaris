# **App Name**: Elysora Digital Safari

## Core Features:

- Interactive Destination Map: Interactive map of Tanzania with key safari destinations, displaying wildlife and attractions for each location.
- Personalized Safari Recommendations: AI powered tool to generate custom safari recommendations based on user preferences such as budget, interests, and travel dates.
- Safari Package Filtering: Advanced filtering for safari packages based on duration, budget, style, destinations, and group size preferences.
- Dynamic Itinerary Generator: Generate detailed safari itineraries, including day-by-day breakdowns with descriptions, activities, meals, and accommodation details. The generator considers travel time.
- High-Quality Media Display: Full-width image galleries and video integration on destination and safari package detail pages, featuring wildlife, landscapes, and safari experiences.
- Real-time Inquiry System: Integration of a WhatsApp float button for quick inquiries and direct communication with Elysora Safaris team.
- Integrated Review Platform: Display and manage customer reviews and testimonials directly on the website with a review submission feature for past customers.

## Style Guidelines:

- Primary color: Terracotta (#D4816F) to evoke the warm earthy tones of the Tanzanian landscape.
- Background color: Beige (#F5F5DC), a desaturated version of terracotta, to create a warm and inviting backdrop.
- Accent color: Forest Green (#2C5F2D) for secondary elements, providing a natural contrast to the warm primary color.
- Headline font: 'Playfair', a serif font, bringing elegance and sophistication. Body font: 'PT Sans', a sans-serif font, which maintains the readability needed for blocks of text.
- Use Lucide React icons for a consistent and modern feel throughout the website.
- Implement card-based layouts for tours and destinations with generous whitespace to enhance readability.
- Subtle animations with Framer Motion for a smooth and engaging user experience. Subtle loading animations guide the user without slowing them down.
